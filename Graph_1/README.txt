README:
This repository contains a graph created using the R program that represents the fatalities rate per 100,000 in Israel from COVID-19 pandemic.
The graph was created using ggplot2 package with additional packages.
The original file of dataset was downloaded from https://ourworldindata.org/coronavirus/country/israel.
The graph also shows the main related COVID-19 events, and two interruption points in March 2020 and February 2021.
